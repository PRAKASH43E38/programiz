Introduction to Java

Java is a high-level, class-based, object-oriented programming language widely used for building enterprise-scale applications.

public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}

Variables and Data Types

In Java, variables store data and must be declared with a specific data type.

int age = 25;
String name = "John";
System.out.println(age);
System.out.println(name);

Control Structures

Control structures like if-else statements let you control program flow based on conditions.

int number = 10;
if (number > 0) {
    System.out.println("Positive");
} else {
    System.out.println("Negative or zero");
}
