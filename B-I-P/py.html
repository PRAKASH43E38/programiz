Introduction to Python

Python is a high-level, interpreted programming language known for its simplicity and readability.

It’s widely used in web development, data science, artificial intelligence, and more.

Basic Syntax

Let’s start with the classic "Hello, World!" program.

print("Hello, World!")

You can run this code in your Python environment to see the output.

Variables and Data Types

In Python, you can declare variables without specifying their type.

age = 25
name = "Alice"
pi = 3.14
is_student = True

Here, age is an integer, name is a string, pi is a float, and is_student is a boolean.

Examples

Here’s a simple example combining variables and print statements.

name = "Bob"
age = 30
print("Hello, my name is", name, "and I am", age, "years old.")

This outputs: Hello, my name is Bob and I am 30 years old.
